# Raspberry Pi Face Recognition Door System

This project was developed in a group of 3 people.

This project implements a smart door system using a Raspberry Pi, camera module, and face recognition. It allows registered users to ring a doorbell, detects their face, and notifies the homeowner via Telegram, giving the option to open the door remotely.

## Features

- Face recognition of registered users
- Doorbell integration with physical button
- Telegram notifications with image and decision buttons (Open Door / Ignore)
- Relay-controlled door lock
- Buzzer and LED feedback for door status
